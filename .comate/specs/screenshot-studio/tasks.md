# Hijessy 截图工具 — 迭代二：Snipaste 风格一体化浮层（任务计划）

> 迭代一（截图 MVP，Task 1–10）已完成，记录见 summary.md。本文件为迭代二 UI/UX 重构任务。

- [✓] Task 1: 主题、依赖与启动模式改造
    - 1.1: 添加依赖 egui-phosphor（图标字体），与 CJK 字体共存安装
    - 1.2: 启动即设 egui::Visuals::light()（Light 主题）
    - 1.3: NativeOptions 改为无边框/透明/置顶/初始不可见（with_visible(false)）
    - 1.4: 启动流程改为直接开启截图会话；不再显示软件主界面
    - 1.5: 会话完成/取消后隐藏窗口(Visible(false)) 常驻，全局快捷键可重新唤起

- [✓] Task 2: overlay 会话骨架（自由框选 + 手柄 + 视觉）
    - 2.1: 由 selection 演进为 overlay 模块，承载全屏冻结遮罩会话
    - 2.2: 选区外整体压暗、选区内保持原亮（Light 视觉，蓝色边框）
    - 2.3: 自由拖拽框选，左上角显示 宽×高 尺寸标签
    - 2.4: 选区四角+四边共 8 个圆点手柄，可二次拖拽调整选区
    - 2.5: Esc 取消、Enter 确认的基础交互

- [✓] Task 3: 窗口自动识别与模式切换
    - 3.1: capture/engine.rs 新增“光标下最上层窗口”查询（命中测试 + z 序）
    - 3.2: 进入浮层默认高亮光标下窗口为选区；移动光标切换高亮
    - 3.3: 「全屏」按钮：一键将选区扩为整屏
    - 3.4: 「窗口」按钮：切回窗口自动识别模式
    - 3.5: 手动拖拽时覆盖窗口模式为自定义选区

- [✓] Task 4: 底部胶囊浮动工具栏（图标 + Tooltip）
    - 4.1: 选区确定后在选区下方外侧渲染圆角胶囊工具栏（白底/圆角/阴影）
    - 4.2: 纯图标按钮（egui-phosphor），hover 显示中文 Tooltip、移开消失
    - 4.3: 工具分组：标注工具组 / 功能组(预留禁用) / 操作组
    - 4.4: 属性弹层：颜色 8 色板、线宽、字号、箭头样式（点开显示，简洁）
    - 4.5: 工具栏随选区位置自适应（下方空间不足则置于选区内下缘）

- [✓] Task 5: 一体化标注与输出闭环
    - 5.1: 在浮层内绘制标注（实时预览 + 提交），坐标映射到冻结图像素
    - 5.2: 复用 compose 合成到冻结全屏，再按选区物理像素裁剪为输出
    - 5.3: 撤销/重做（含快捷键）在浮层内可用
    - 5.4: 确认✓→复制剪贴板、保存⬇→落盘、取消✕/Esc→丢弃
    - 5.5: 完成后隐藏窗口常驻；快捷键可重开会话

- [✓] Task 6: 联调、Light 主题打磨与体积复核
    - 6.1: 窗口识别 + 模式切换 + 框选 + 标注 + 输出端到端联调
    - 6.2: Light 主题细节（配色/阴影/间距/图标对齐）打磨
    - 6.3: 边界复核（Wayland 退化为手动框选、命中不到窗口默认全屏、空选区取消）
    - 6.4: cargo build --release 通过并复核二进制体积，清理临时文件
